# Pokedex — Next.js Web

A simple, responsive Pokedex web app built with **Next.js**.  
Browse Pokémon and view details with pagination — a small demo project to practice React/Next.js and API integration.

Live demo: https://pokedex-web-phi.vercel.app/ :contentReference[oaicite:1]{index=1}

---

## Features
- List of Pokémon with pagination (you can place a like on any pokemon you want)
- Individual Pokémon detail pages

---

## Tech stack
- Next.js (React)
- Fetch (API calls)
- CSS using Bootstrap
- Deployed on Vercel

---

## Run locally

```bash
# clone
git clone https://github.com/Baizho/Pokedex-nextjs-web.git
cd Pokedex-nextjs-web

# install
npm install

# dev
npm run dev
# open http://localhost:3000
